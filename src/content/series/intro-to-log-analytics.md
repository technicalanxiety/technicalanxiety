## Overview

Azure Log Analytics is the foundation of Azure monitoring. This series covers the essentials: writing KQL queries, understanding log structure, and building operational insights.

## What You'll Learn

### KQL Fundamentals
- Query syntax and structure
- Filtering, aggregating, and joining data
- Time-based analysis
- Common query patterns

### Log Analytics Workspace
- Workspace design and organization
- Data retention and costs
- Connecting data sources
- Query performance

### Practical Applications
- Troubleshooting with logs
- Building operational dashboards
- Alert rules and automation
- Best practices for production use

## Who This Is For

- **Operations Teams** new to Azure monitoring
- **Engineers** learning KQL
- **Anyone** starting with Log Analytics
- **Teams** building Azure observability
